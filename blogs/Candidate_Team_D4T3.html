本质上CDSF构造了一个的方案，给这个灯标号，Alice将亮着的标号都异或起来，与目标数再异或一下就是需要调整的灯的编号。

一个优秀的88分构造

很显然我们可以优化一下CDSF的做法。

号灯显然可以使用“不进行操作”代替，所以需要的灯的数量变成了，本质上是一个的方案。

代码非常简洁。

 
for (int i = 0; i < 1 << n; i++) {
    int f = 0;
    for (int j = 0; j < n; j++) {
        if ((i >> j) & 1) {
            f ^= j + 1;
        }
    }
    printf("%d ", f & 7);
}

可以过后个点，期望得分。
